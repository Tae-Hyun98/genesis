## 1. 프로젝트 소개
- genesis를 프로모션형태로 변경하여 제작한 프로모션사이트입니다.

- 퍼블리싱 및 기능구현은 **개인 100%** 작업입니다.

- HTML5, SCSS로 마크업 및 스타일을 입혔으며, gsap.js라이브러리를 사용하여 스크롤애니메이션효과를 주었습니다.

- 주요기능으로는 **상품검색, 상품필터, 리뷰, 상품문의, 로그인/회원가입 유효성체크** 등이 있습니다.
